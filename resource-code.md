Our group develops or has previously developed several open-source software packages for electron microscopy simulation, data analysis, and materials characterization.

## quantEM

::::{div}
<div style="width: 60%; float: right; margin-left: 1em;"><img src="/images/research/quantem_logo_53.png" alt="quantEM" style="width: 100%;" /></div>

[quantEM](https://github.com/electronmicroscopy/quantem) is a unified Python toolkit for quantitative electron microscopy analysis. It provides a modular framework for processing, simulating, and analyzing electron microscopy data, with GPU acceleration via CuPy and PyTorch.

::::

- [GitHub Repository](https://github.com/electronmicroscopy/quantem)

## py4DSTEM

::::{div}
<div style="width: 15%; float: right; margin-left: 1em;"><img src="/images/research/py4DSTEM_logo_54_export.png" alt="py4DSTEM" style="width: 100%;" /></div>

[py4DSTEM](https://github.com/py4dstem/py4DSTEM) is an open-source Python package for analysis of four-dimensional scanning transmission electron microscopy (4DSTEM) datasets. It supports a wide range of methods including orientation and phase mapping, strain mapping, fluctuation electron microscopy, differential phase contrast, parallax imaging, and ptychography.

::::

- [GitHub Repository](https://github.com/py4dstem/py4DSTEM)
- [Tutorials](https://github.com/py4dstem/py4DSTEM_tutorials)
- [Google Colab Tutorials](http://tinyurl.com/py4DSTEM)

In 2023, py4DSTEM was awarded an [R&D100 award](https://www.rdworldonline.com/rd-100-2023-winner/py4dstem/)!

## Prismatic

[Prismatic](https://github.com/prism-em/prismatic) is a GPU-accelerated simulation code for scanning and high resolution transmission electron microscopy (STEM and HRTEM). It implements the PRISM algorithm developed by Colin, which achieves speedups of f^2^ to f^4^ compared to conventional multislice methods, enabling efficient large-scale simulations.

- [GitHub Repository](https://github.com/prism-em/prismatic)
- [Documentation](https://prism-em.com/)
- [Prismatic 2.0 paper](doi.org/10.1016/j.micron.2021.103141)
